# Health Clinic Availability App

This is a code bundle for Health Clinic Availability App. The original project is available at https://www.figma.com/design/DhNboFWDHDQQeR13VsFLzX/Health-Clinic-Availability-App.

## Running the code

1. Run `npm i` to install dependencies.
2. Copy `.env.example` to `.env` and update PostgreSQL credentials as needed.
3. Start the backend API with `npm run server`.
4. In another terminal, start the frontend with `npm run dev`.

The API uses PostgreSQL and auto-creates the `users` table on server startup.
